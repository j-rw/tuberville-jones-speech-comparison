Methodology

  • All topic/word/name comparisons use per-1,000-word rates.
  • Top-word tables use the same substantive filter for both candidates: drop function words, spoken fillers, generic English (little, every, big, believe, country, etc.), known interview host names, and each candidate's own first/last name tokens.
  • Each candidate's frequency tables are sorted descending by that candidate's /1k rate.
  • video_share_pct = % of appearances in window mentioning the topic.
  • rate_per_1000 = count / total_words * 1000
  • Per video: count unique substantive word types, divide by that video's total words, multiply by 1,000. Candidate score = median across appearances (one vote per video).
  • Fair length control: one random 7-minute window on each candidate's speech timeline per appearance (reproducible seed). Clips shorter than 7 minutes of speech use all speech.
